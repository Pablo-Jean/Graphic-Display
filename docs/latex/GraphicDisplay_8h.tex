Main header file of Graphic Display Library, here we have the middle layer of the library. Where we works on drawing elements (texts, bitmaps and geometric forms). In drivers folder we have the low layer, composed by display controller drivers. The drivers are independet of the Graphic Display files, you can use them in more advanced libraries, like \href{https://github.com/lvgl/lvgl}{\texttt{ LVGL}}. This library is created to be flexible and scalable, allowing the engineer to use any amount of display, of any type.
